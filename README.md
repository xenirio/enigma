1) Install Angular CLI  
> `npm install -g @angular/cli`

2) Install Packages
> `npm install`

3) Start Site
> `ng serve`

4) Install Angular Bootstrap (https://github.com/valor-software/ngx-bootstrap)
> `ng add ngx-bootstrap`
